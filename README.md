# BlinkCare

BlinkCare is a lightweight desktop utility that reminds you to blink regularly and reduce eye strain while working on a computer.

## Features

- Floating blink reminder widget
- Circular countdown timer
- System tray integration
- Pause and resume functionality
- Configurable work interval and blink duration
- Draggable popup with saved position
- Automatic startup with Windows

## Why BlinkCare?

Staring at screens reduces blink rate, leading to dry eyes and eye strain. BlinkCare helps maintain healthy blinking habits with gentle reminders.
